The basics

Don't forget to "return" if the goal is to return something

To run a ruby file in the shell => ruby "filename".rb

To test a ruby file in the shell => rspec "filename".rb

To test the code => type "irb" in the shell, to exit, "ctrl d"

Good to Know

Variables starts with lowercase

"print" is horizontal, "puts" is vertical

"puts" means put string

"gets" means retrieve string

"gets.chomp" takes what's given

capital letter has precedence over lowercase => cat < Xander ... false, cat < Xander.downcase ...true

objects: string, integer, float

\t => prints a tab

recurrent syntactic sugar => use the "+" sign

Methods

to call a method: type its name with the parameters in parenthesis

to create a method: def "name"(arg1,arg2), (block of code), end

methods: puts, gets, chomp, to_i, to_s, to_f, +, -, *, ?, % => every methods need an object

a method ending with ? => returns a Boolean

a method ending with ! => will change the original object

A block of code inside a method can be delimited by {}

Flow control

Flow control => 1) conditional, 2) looping, 3) iteration, 4) exception

  • conditional: if....; then.....;end or, if... elsif....elsif....else...end
  • conditional: unless....(block of code) end
  • conditional: case name when ... then... when... then... else... end
  • looping: while ... (block of code) end
  • looping: until... (block of code) end
  • looping: for ... in ... (block of code) end
  • looping: loop do ...... end
  • iterators: .times, .each, .map (works with array)=> always foll'd by block of code
  • exception: 1)raise, 2) rescue, 3) ensure

Classes and variables

to create a class => class Capital_letter def initialize () @class_variable end

#global_variable = variable1 => retrievable anywhere

@class variable = variable2 => retievable inside the class only

Arrays

to create an array: a = Array.new, a = Array.new(2) => specifies the size, a = Array.new(3, "abc") => ["abc", "abc, "abc"], array = []

also to create an array: %w{string} => ["s","t","r","i","n","g"], %i{a,b,c} =>[:a, :b, :c] => creates an array of symbols

to retrieve an element in an array: array[index#]

Hashes

to create a hash: h = Hash.new, h = Hash.new(0) => gives a default value of 0, h = {}, Hash {"Florida"=> "FL", "Delaware"=> "DE"}, Hash[[1,2],[3,4]] =>{1=>2, 3=>4}

to retrieve a value in a hash: hash["key"], hash.fetch("key"), hash.values_at("key", "key")
